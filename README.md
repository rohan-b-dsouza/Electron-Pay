# ElectronPay

A peer-to-peer (P2P) digital payment web application that enables users to send money using usernames.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB

## Setup

```bash
# Clone the repository
git clone https://github.com/your-username/electronpay.git

# Backend setup
cd electronpay/backend
npm install
node index.js

# Frontend setup (in a new terminal)
cd electronpay/frontend
npm install
npm run dev
